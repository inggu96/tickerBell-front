| 마이페이지    | - 회원가입시 설정한 권한에 따라 예매자 -> 이벤트 예매내역 , <br>이벤트 등록자 -> 이벤트 등록내역 구현 <br> - 비밀번호 수정 기능 구현 비밀번호 확인 api를 호출 후 기존 비밀번호와 확인 후 새 비밀번호 지정가능                                                                              |
